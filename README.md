Game made to run on a custom hardware with raspberry pi 5 rig with cameras mounted. 
Inputs via keyboard and camera with AI-models trained to classify dice and cards to this digital boardgame.

Game is the main game.
Interface handles capturing cards and dice and sending info to the game. 
Teacher runs locally on desktop/better hardware to train models.
